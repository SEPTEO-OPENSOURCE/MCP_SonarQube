## Introduction

Un outil MCP (Model Context Provider) est une fonctionnalit√© expos√©e via le protocole MCP qui permet √† un assistant IA (comme Cursor) d'interagir avec SonarQube.

**Flux d'un outil MCP** :
```

                                                               ‚Üì
Cursor IDE ‚Üê MCP Server ‚Üê CommandResult ‚Üê Commands ‚Üê API ‚Üê SonarQube
```
